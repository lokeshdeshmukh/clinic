# ClinicFlow Appointment Booking System

ClinicFlow is a mobile-first appointment booking platform for small clinics built with PHP 8.3+, MySQL 8+, Tailwind CSS, JavaScript, FullCalendar, PHPMailer, and a lightweight MVC architecture.

The application supports:

- Clinic admin registration, email verification, login, password reset
- Patient registration, login, booking, cancellation, rescheduling
- Doctor management, availability scheduling, blocked slots, holidays
- Double-booking protection with transactional locking
- Appointment, revenue, and analytics dashboards
- CSV, Excel-compatible, and PDF exports
- REST API endpoints for clinics, doctors, slots, and appointments

Shared-hosting deployment is supported with ZIP upload plus a browser installer. See [INSTALL.md](INSTALL.md).
